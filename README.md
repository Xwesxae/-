P2P (Peer-to-Peer) архитектура:
text
Каждый клиент = Сервер + Клиент
├── UDP сервер (порт 12345) - прием сообщений
├── UDP broadcast (порт 12346) - обнаружение

Инструкция для разработчика

python

├── __init__() - инициализация
